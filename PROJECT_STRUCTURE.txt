KEY FILES TO CONFIGURE:
-----------------------
1. .env.local                   вҶ’ Create with VAPID keys
2. public/icon-*.png            вҶ’ Replace with your icons
3. scripts/cron-scheduler.js    вҶ’ Change notification time (line 30)
4. app/api/push/send-daily/     вҶ’ Customize notification message

QUICK START:
-----------
1. Create .env.local (see START_HERE.md)
2. npm run dev
3. Open http://localhost:3000
4. Enable notifications and test!

DAILY NOTIFICATIONS:
-------------------
npm run cron (in a separate terminal)

DOCUMENTATION:
-------------
Start with START_HERE.md or INDEX.md
